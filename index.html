Etiqueta Descripcion
<html> Para iniciar el html.
<base> Definir el URL
<head> Para establecer metadatos del html
<link>> Enlaces, pero usualmente se usan en hojas de estilos
<meta> Para colocar información del documento html
<style> Define estilos directamente: <style> y </style> (ambas obligatorias).
<title> Metadato de titulo que va en head
<body> Es el cuerpo del html, basicamente lo que se puede ver
<address> Agrega información a la raiz principal o a la raiz padre
<article> Composicion que puede ser reutilizable en el html
<aside> Consiste en contenido relacionado con el contenido principal del documento
<footer> Cuerpo de abajo del html
<header> Elementos de arriba del html
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Titulos preestablecidos
<main> contendido principal del html
<nav> Navegar en el documento por links
<section> para definir secciones genericos
<blockquote> crear referencias por medio de bloques
<dd> proporciona detalles de un elemento anterior, usando listas
<div> etiqueta de bloque completo
<dl> lista descriptiva
<dt> elemento descriptivo en una list
<figcaption> subtitulo
<figure> titulo independiente
<hr> cambio de parrafo
<li> elemento de lista
<menu> lista desordenada
<ol> viñetas ordenadas
<p> texto de parrafo
<pre> muesta el elemento de preformateado, diferente de lo que esta en html
<ul> lista desordenada
<a> enlace a otras paginas
<abbr> abrebiatura que se muestra al pasar el cursor
<b> texto importante
<bdi> aisla un elemento de un grupo preformateado
<bdo> anula al algoritmo bidimencional
<br> saltar de parrafo
<cite> cita corta
<code> muestra el codigo tal y como esta
<data> convierte para que sea interpretable por la maquina
<dfn> marca termino por definicion
<em> texto importante pero no tanto
<i> texto importante pero menos que el anterior
<kbd> marca tecla
<mark> texto resaltado
<q> cita corta
<rp> (en-US) parentesis de imporancia, remplaza lo que se vio antes
<rt> (en-US) elemento ruby, en texto ruby
<ruby> (en-US) texto al lado de main
<s> texto destacado
<samp> ejemplo de salida de programa
<small> cambio del tamaño
<span> bloque de código, acomodado
<strong> elemento importante
<sub> subindice
<sup> superindice
<time> para elementos de tiempo
<u> subrayar texto
<var> variables de programa
<wbr> salto de línea si se encoge la pantalla
<area> dividir el body en varios campos
<audio> Insertar audio
<img> imagen
<map> describir uno o dos parrafos
<track> hijo de elemento multimedia
<video> para video
<caption> titulo de tabla
<col> clocar datos para columna
<colgroup> para crear la tabla
<table> para el cuerpo de tabla
<tbody> (en-US) para el cuerpo de tabla
<td> elemento de tabla
<tfoot> (en-US) elementos de abajo tabla, definiciones
<th> encabezado de celda
<thead> (en-US) para atributos de tabla
<tr> define tuplas
<button> crear botones marcados
<datalist> crear lista de datos que se usarán
<fieldset> permite dividir por grupos en diferentes campos el formulario
<form> crear formulario
<input> para ingresar valores de entrada
<label> etiqueta
<legend> titulo para un grupo de campos
<meter> (en-US) valor escalar
<optgroup> (en-US) grupo dentro de select
<option> representar en un formulario un elemento en select o optgroup
<output> (en-US) donde se puede representar los resultados
<progress> Para mostrar el avance de una tarea
<select> muestra un menú de opciones
<textarea> edición multilinea del texto sin formato
